# 장난감 조립

## 1. 문제 요약

완제품 번호는 `N`이고, 부품 관계 `X Y K`는
"부품 `X`를 만들 때 부품 `Y`가 `K`개 필요하다"를 의미합니다.

부품은 기본 부품과 중간 부품으로 나뉩니다.
- 기본 부품: 다른 부품으로 더 쪼갤 수 없는 부품
- 중간 부품: 기본/중간 부품을 조합해서 만드는 부품

목표는 완제품 `N` 1개를 만들기 위해 필요한 **기본 부품별 개수**를 구하는 것입니다.

---

## 2. 입력 조건 정리

- 부품 개수 `N` (`3 <= N <= 100`)
- 관계 개수 `M` (`3 <= M <= 100`)
- 다음 `M`줄: `X Y K`
  - `X`를 만드는 데 `Y`가 `K`개 필요
- `1`번부터 `N-1`번은 기본/중간 부품 번호, `N`은 완제품 번호
- 두 중간 부품이 서로를 필요로 하는 순환 관계는 없습니다.

---

## 3. 출력 조건 정리

- 완제품 `N`을 만들 때 필요한 기본 부품만 출력합니다.
- 기본 부품 번호 오름차순으로 출력합니다.
- 형식: `부품번호 개수`
