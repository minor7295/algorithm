# 같은 나머지 (BOJ 1684) 문제 분석

## 1) 문제 요약
정수 수열 `N_1, N_2, ..., N_n`이 주어졌을 때,
"모든 수를 같은 `D`로 나눴을 때 나머지가 같아지는" 경우의 `D`들 중 최댓값을 구하는 문제입니다.

각 원소를 다음처럼 표현할 수 있습니다.

- `N_i = D * Q_i + R`

여기서
- `D`: 우리가 찾아야 하는 나눗수
- `Q_i`: 각 원소의 몫
- `R`: 모든 원소에서 공통으로 나오는 나머지

## 2) 처음에 막히는 지점
이 문제는 미지수가 많습니다.

- `D`를 모름
- `Q_i`를 모름
- `R`도 모름

그래서 `D`를 1부터 전부 대입하는 완전탐색은 비효율적이고,
`R`이 어떤 값인지 모르는 상태라 일반적인 점화식 접근도 바로 적용하기 어렵습니다.

## 3) 입력/출력 조건
- `1 <= n <= 1,000`
- 각 정수의 절댓값은 `1,000,000` 이하
- 항상 정답(가장 큰 `D`)이 존재하는 입력만 주어짐
- 출력: 조건을 만족하는 가장 큰 `D`
