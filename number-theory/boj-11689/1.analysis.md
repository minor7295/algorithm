# GCD(n, k) = 1

## 📌 문제 요약

자연수 n이 주어졌을 때, GCD(n, k) = 1을 만족하는 자연수 1 ≤ k ≤ n 의 개수를 구하는 프로그램을 작성하시오.

---

## 🔍 문제 설명

* **목표**: 1부터 n까지의 자연수 중에서 n과 서로소인 수의 개수 구하기
* **서로소**: 두 수의 최대공약수가 1인 경우 (GCD(a, b) = 1)
* **예시**: 
  - n = 5일 때, GCD(5, 1) = 1, GCD(5, 2) = 1, GCD(5, 3) = 1, GCD(5, 4) = 1, GCD(5, 5) = 5
  - 서로소인 수: 1, 2, 3, 4 → 총 4개

---

## 📥 입력 조건

* 첫째 줄에 자연수 n (1 ≤ n ≤ 10¹²)이 주어진다.

---

## 📤 출력 조건

* GCD(n, k) = 1을 만족하는 자연수 1 ≤ k ≤ n 의 개수를 출력한다.

---

## 💡 문제 핵심 해석

이 문제는 **오일러 파이 함수(Euler's Totient Function) φ(n)**를 구하는 문제입니다.

> 📖 **오일러 파이 함수 상세 설명**: [수론(Number Theory) 정리](../number-theory.md#3-오일러-파이-함수-eulers-totient-function)

**문제 요구사항:**
- "GCD(n, k) = 1을 만족하는 자연수 1 ≤ k ≤ n 의 개수"
- 이것은 정확히 오일러 파이 함수 φ(n)의 정의와 일치합니다.

**예시:**
- φ(1) = 1
- φ(5) = 4 (1, 2, 3, 4)
- φ(10) = 4 (1, 3, 7, 9)
- φ(45) = 24
- φ(99) = 60

---

## ✨ 예시

### 예시 1

**입력**
```
1
```

**출력**
```
1
```

**설명**: 
* GCD(1, 1) = 1
* 서로소인 수: 1 → 총 1개

### 예시 2

**입력**
```
5
```

**출력**
```
4
```

**설명**: 
* GCD(5, 1) = 1 ✓
* GCD(5, 2) = 1 ✓
* GCD(5, 3) = 1 ✓
* GCD(5, 4) = 1 ✓
* GCD(5, 5) = 5 ✗
* 서로소인 수: 1, 2, 3, 4 → 총 4개

### 예시 3

**입력**
```
10
```

**출력**
```
4
```

**설명**: 
* 10의 소인수: 2, 5
* GCD(10, 1) = 1 ✓
* GCD(10, 2) = 2 ✗
* GCD(10, 3) = 1 ✓
* GCD(10, 4) = 2 ✗
* GCD(10, 5) = 5 ✗
* GCD(10, 6) = 2 ✗
* GCD(10, 7) = 1 ✓
* GCD(10, 8) = 2 ✗
* GCD(10, 9) = 1 ✓
* GCD(10, 10) = 10 ✗
* 서로소인 수: 1, 3, 7, 9 → 총 4개

### 예시 4

**입력**
```
45
```

**출력**
```
24
```

**설명**: 
* 45 = 3² × 5
* φ(45) = 45 × (1 - 1/3) × (1 - 1/5) = 45 × (2/3) × (4/5) = 24

### 예시 5

**입력**
```
99
```

**출력**
```
60
```

**설명**: 
* 99 = 3² × 11
* φ(99) = 99 × (1 - 1/3) × (1 - 1/11) = 99 × (2/3) × (10/11) = 60

---

## 📝 정리

이 문제는 **오일러 파이 함수 φ(n)을 계산하는 문제**로, n의 소인수 분해를 통해 효율적으로 계산할 수 있습니다.

시간 복잡도 O(√n)로 해결할 수 있으며, n ≤ 10¹²이므로 충분히 효율적입니다.
