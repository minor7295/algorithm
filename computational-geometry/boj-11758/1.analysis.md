# CCW (Counter Clockwise)

## 📌 문제 요약

2차원 좌표 평면 위에 있는 점 3개 P1, P2, P3가 주어진다. P1, P2, P3를 순서대로 이은 선분이 어떤 방향을 이루고 있는지 구하는 프로그램을 작성하시오.

---

## 🔍 문제 설명

* 2차원 좌표 평면 위의 세 점 P1, P2, P3가 주어진다
* 세 점을 순서대로 이었을 때의 방향을 판단해야 한다
* 반시계 방향이면 1, 시계 방향이면 -1, 일직선이면 0을 출력한다

---

## 📥 입력 조건

* 첫째 줄에 P1의 (x1, y1)이 주어진다
* 둘째 줄에 P2의 (x2, y2)가 주어진다
* 셋째 줄에 P3의 (x3, y3)가 주어진다
* 모든 좌표는 정수이며, -10,000 ≤ x1, y1, x2, y2, x3, y3 ≤ 10,000
* P1, P2, P3의 좌표는 서로 다르다

---

## 📤 출력 조건

* P1, P2, P3를 순서대로 이은 선분이 반시계 방향을 나타내면 **1**
* 시계 방향이면 **-1**
* 일직선이면 **0**을 출력한다

---

### 📌 핵심 제약 조건

1. **세 점의 방향 판단**
   - 반시계 방향: P1 → P2 → P3가 반시계 방향으로 회전
   - 시계 방향: P1 → P2 → P3가 시계 방향으로 회전
   - 일직선: 세 점이 한 직선 위에 있음

2. **좌표 범위**
   - 모든 좌표는 정수
   - 범위: -10,000 ~ 10,000
   - 오버플로우 주의 필요 (int 범위 내에서 해결 가능)

**예시 분석:**
```
예제 1: P1(1,1), P2(5,5), P3(7,3)
→ 시계 방향 (-1)

예제 2: P1(1,1), P2(3,3), P3(5,5)
→ 일직선 (0)

예제 3: P1(1,1), P2(7,3), P3(5,5)
→ 반시계 방향 (1)
```

---

## ✨ 예시

### 예시 1

**입력**
```
1 1
5 5
7 3
```

**출력**
```
-1
```

**설명**: 
* P1(1,1) → P2(5,5) → P3(7,3)는 시계 방향으로 회전
* 시계 방향이므로 -1 출력

### 예시 2

**입력**
```
1 1
3 3
5 5
```

**출력**
```
0
```

**설명**: 
* P1(1,1), P2(3,3), P3(5,5)는 모두 한 직선 위에 있음
* 일직선이므로 0 출력

### 예시 3

**입력**
```
1 1
7 3
5 5
```

**출력**
```
1
```

**설명**: 
* P1(1,1) → P2(7,3) → P3(5,5)는 반시계 방향으로 회전
* 반시계 방향이므로 1 출력

---

## 📝 정리

이 문제는 **세 점의 방향을 판단하는 계산 기하학 문제**로, 외적(Cross Product)을 활용하여 해결할 수 있다.

**문제의 특징:**
- 입력이 매우 작음 (세 점의 좌표만)
- 시간 복잡도 O(1)로 해결 가능
- 계산 기하학의 기본 문제로, CCW 알고리즘을 학습하기에 적합

**핵심 포인트:**
- 세 점을 순서대로 이었을 때의 방향 판단
- 반시계 방향(1), 시계 방향(-1), 일직선(0) 세 가지 경우
- 수학적 도구(외적)를 활용한 정확한 판단 필요
