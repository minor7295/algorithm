# 화살표 그리기

## 📌 문제 요약

직선 위에 N개의 점이 주어지고, 각 점은 색깔을 가진다. 각 점에서 같은 색깔을 가진 다른 점들 중 가장 가까운 점으로 화살표를 그릴 때, 모든 화살표의 길이 합을 구하라.

---

## 📥 입력 조건

* 첫 번째 줄에는 점들의 개수를 나타내는 정수 N이 주어진다.
* 다음 N개의 줄 각각에는 점의 좌표와 색깔을 나타내는 두 정수 x와 y가 주어진다.
* `2 ≤ N ≤ 10⁵`
* `0 ≤ x ≤ 10⁵`, `1 ≤ y ≤ N`

---

## 📤 출력 조건

* 모든 점에서 시작하는 화살표들의 길이 합을 출력한다.

---

## 📌 핵심 제약 조건

1. **각 점은 같은 색깔을 가진 다른 점이 항상 존재함**
   - 모든 점에 대해 화살표를 그릴 수 있음

2. **가장 가까운 점이 두 개 이상이면 아무거나 하나를 선택**
   - 거리가 같으면 아무거나 선택 가능

3. **두 점 사이의 거리는 위치의 차이**
   - `|x₁ - x₂|`

---

## ✨ 예시

### 예시 1

**입력**
```
5
0 1
1 2
3 1
4 2
5 1
```

**출력**
```
13
```

**설명**: 
* 색깔 1: 위치 [0, 3, 5]
  * 점 0 → 점 3 (거리: 3)
  * 점 3 → 점 5 (거리: 2) 또는 점 0 (거리: 3) → 점 5 선택 (거리: 2)
  * 점 5 → 점 3 (거리: 2)
* 색깔 2: 위치 [1, 4]
  * 점 1 → 점 4 (거리: 3)
  * 점 4 → 점 1 (거리: 3)
* 합: 3 + 3 + 2 + 3 + 2 = 13

### 예시 2

**입력**
```
7
6 1
7 2
9 1
10 2
0 1
3 1
4 1
```

**출력**
```
16
```
