# 이친수

## 📌 문제 요약

0과 1로만 이루어진 수 중에서 다음 두 가지 성질을 만족하는 수를 **이친수(pinary number)**라고 한다:

1. 이친수는 0으로 시작하지 않는다.
2. 이친수에서는 1이 두 번 연속으로 나타나지 않는다. (즉, 11을 부분 문자열로 갖지 않는다)

N(1 ≤ N ≤ 90)이 주어졌을 때, N자리 이친수의 개수를 구하는 프로그램을 작성하시오.

---

## 🔍 문제 설명

* N자리 이진수 중에서 이친수 조건을 만족하는 것의 개수를 구해야 한다
* 조건 1: 첫 자리는 반드시 1이어야 함 (0으로 시작 불가)
* 조건 2: 1이 연속으로 두 번 나타나면 안 됨 (11 불가)

**예시:**
* 1자리: 1 (1개)
* 2자리: 10 (1개)
* 3자리: 100, 101 (2개)
* 4자리: 1000, 1001, 1010 (3개)
* 5자리: 10000, 10001, 10010, 10100, 10101 (5개)

---

## 📥 입력 조건

* 첫째 줄에 N (1 ≤ N ≤ 90)이 주어진다

---

## 📤 출력 조건

* 첫째 줄에 N자리 이친수의 개수를 출력한다

---

## 💡 문제 핵심 해석

이 문제는 작은 예시를 나열하여 규칙을 발견하면 점화식으로 해결할 수 있는 문제입니다.

**작은 예시로 규칙 찾기:**

```
1자리: 1 (1개)
2자리: 10 (1개)
3자리: 100, 101 (2개)
4자리: 1000, 1001, 1010 (3개)
5자리: 10000, 10001, 10010, 10100, 10101 (5개)
```
---

## 📌 핵심 제약 조건

1. **첫 자리는 반드시 1**: 0으로 시작하는 수는 이친수가 아님
2. **1이 연속으로 두 번 나타나면 안 됨**: 11을 부분 문자열로 가질 수 없음
3. **N의 범위**: 1 ≤ N ≤ 90 (피보나치 수열이므로 값이 매우 큼 → long 타입 필요)

**예시 분석:**
```
✅ 이친수: 1, 10, 100, 101, 1000, 1001, 1010
❌ 이친수 아님: 0010101 (0으로 시작), 101101 (11 포함)
```

➡️ **동적 계획법(DP)**을 사용하여 해결할 수 있다.
