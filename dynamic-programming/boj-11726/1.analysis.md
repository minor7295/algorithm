# 2×n 타일링

## 📌 문제 요약

2×n 크기의 직사각형을 1×2, 2×1 타일로 채우는 방법의 수를 구하는 문제입니다.

**예시:**
- n = 2인 경우:
  - 세로 타일 2개 (2×1 타일 2개)
  - 가로 타일 2개 (1×2 타일 2개)
  - 총 2가지 방법

- n = 3인 경우:
  - 세로 타일 3개
  - 세로 타일 1개 + 가로 타일 2개 (2가지 배치)
  - 총 3가지 방법

---

## 🔍 문제 설명

* 2×n 크기의 직사각형을 1×2(가로), 2×1(세로) 타일로 채워야 합니다
* 타일을 회전시킬 수 있습니다
* 타일을 겹치거나 벗어나게 놓을 수 없습니다

**예시 분석:**
* n = 1: 세로 타일 1개 (1가지)
* n = 2: 세로 타일 2개, 가로 타일 2개 (2가지)
* n = 3: 세로 타일 3개, (세로 1개 + 가로 2개) × 2가지 (3가지)
* n = 4: 5가지 방법
* n = 5: 8가지 방법

**규칙 발견:**
- n = 1: 1
- n = 2: 2
- n = 3: 3
- n = 4: 5
- n = 5: 8
- **피보나치 수열과 유사한 패턴!**

---

## 📥 입력 조건

* 첫째 줄에 n이 주어진다
* 1 ≤ n ≤ 1,000

---

## 📤 출력 조건

* 2×n 크기의 직사각형을 채우는 방법의 수를 10,007로 나눈 나머지를 출력한다

---

## 📌 핵심 제약 조건

1. **타일 종류**: 1×2(가로), 2×1(세로) 타일만 사용 가능
2. **타일 배치**: 타일을 겹치거나 벗어나게 놓을 수 없음
3. **모듈로 연산**: 결과를 10,007로 나눈 나머지를 출력해야 함
4. **n의 범위**: 1 ≤ n ≤ 1,000 (DP로 효율적으로 해결 가능)
