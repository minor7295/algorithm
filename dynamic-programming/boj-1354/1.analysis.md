# 무한 수열 2 (BOJ 1354) 문제 분석

## 1) 문제 요약
수열 `A`는 다음 규칙으로 정의됩니다.

- `A_i = 1` (`i <= 0`)
- `A_i = A_(floor(i / P) - X) + A_(floor(i / Q) - Y)` (`i >= 1`)

입력으로 `N, P, Q, X, Y`가 주어질 때 `A_N`을 구합니다.

## 2) 입력 조건
- `0 <= N <= 10^13`
- `2 <= P, Q <= 10^9`
- `0 <= X, Y <= 10^9`

## 3) 출력 조건
- `A_N` 값을 출력합니다.
