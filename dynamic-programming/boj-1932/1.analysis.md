# 정수 삼각형

## 📌 문제 요약

맨 위층부터 시작해서 아래에 있는 수 중 하나를 선택하여 아래층으로 내려올 때, 이제까지 선택된 수의 합이 최대가 되는 경로를 구하는 프로그램을 작성하라.

아래층에 있는 수는 현재 층에서 선택된 수의 대각선 왼쪽 또는 대각선 오른쪽에 있는 것 중에서만 선택할 수 있다.

---

## 🔍 문제 설명

* 정수로 이루어진 삼각형이 주어진다
* 맨 위층부터 시작하여 아래로 내려간다
* 각 층에서 대각선 왼쪽 또는 대각선 오른쪽으로만 이동 가능
* 경로의 합이 최대가 되는 값을 구해야 한다

---

## 📥 입력 조건

* 첫째 줄에 삼각형의 크기 n(1 ≤ n ≤ 500)이 주어진다
* 둘째 줄부터 n+1번째 줄까지 정수 삼각형이 주어진다
* 삼각형을 이루고 있는 각 수는 모두 정수이며, 범위는 0 이상 9999 이하이다

---

## 📤 출력 조건

* 첫째 줄에 합이 최대가 되는 경로에 있는 수의 합을 출력한다

---

## 📌 핵심 제약 조건

1. **대각선 이동만 가능**
   - 현재 위치 (i, j)에서 아래로 내려갈 때
   - 왼쪽: (i+1, j)
   - 오른쪽: (i+1, j+1)
   - 이 두 위치로만 이동 가능

2. **각 위치마다 도달 가능한 경로가 다름**
   - 왼쪽 끝 위치: 왼쪽 위에서만 올 수 있음
   - 오른쪽 끝 위치: 오른쪽 위에서만 올 수 있음
   - 중간 위치: 양쪽 위에서 올 수 있음

3. **최대 합을 구해야 한다**
   - 모든 가능한 경로 중 합이 최대인 경로

---

## ✨ 예시

### 예시 1

**입력**
```
5
7
3 8
8 1 0
2 7 4 4
4 5 2 6 5
```

**출력**
```
30
```

**설명**: 
* 최적 경로: 7 → 3 → 8 → 7 → 5
* 합: 7 + 3 + 8 + 7 + 5 = 30
