# 최대 힙

## 📌 문제 요약

널리 잘 알려진 자료구조 중 최대 힙이 있다. 최대 힙을 이용하여 다음과 같은 연산을 지원하는 프로그램을 작성한다:

1. 배열에 자연수 x를 넣는다.
2. 배열에서 가장 큰 값을 출력하고, 그 값을 배열에서 제거한다.

프로그램은 처음에 비어있는 배열에서 시작하게 된다.

---

## 🔍 문제 설명

* 자연수 x가 입력되면 배열에 추가한다.
* 0이 입력되면 배열에서 가장 큰 값을 출력하고 제거한다.
* 배열이 비어있을 때 0을 출력하라고 하면 0을 출력한다.

---

## 📥 입력 조건

* 첫째 줄에 연산의 개수 N(1 ≤ N ≤ 100,000)이 주어진다.
* 다음 N개의 줄에는 연산에 대한 정보를 나타내는 정수 x가 주어진다.
  * x가 자연수라면 배열에 x라는 값을 넣는(추가하는) 연산이다.
  * x가 0이라면 배열에서 가장 큰 값을 출력하고 그 값을 배열에서 제거하는 경우이다.
* 입력되는 자연수는 2^31보다 작다.

---

## 📤 출력 조건

* 입력에서 0이 주어진 횟수만큼 답을 출력한다.
* 만약 배열이 비어 있는 경우인데 가장 큰 값을 출력하라고 한 경우에는 0을 출력하면 된다.

---

## 💡 문제 핵심 해석

이 문제는 **최대 힙(Max Heap)** 자료구조를 구현하는 문제이다.

* **최대 힙**: 부모 노드가 자식 노드보다 항상 크거나 같은 완전 이진 트리
* **핵심 연산**:
  * **push(x)**: 자연수 x를 힙에 추가
  * **pop()**: 힙의 최댓값(루트 노드)을 반환하고 제거
* **빈 힙 처리**: 힙이 비어있을 때 pop 연산을 수행하면 0을 출력

예를 들어, 입력이 다음과 같다면:
```
13
0      → 힙이 비어있으므로 0 출력
1      → 힙에 1 추가
2      → 힙에 2 추가
0      → 최댓값 2 출력 및 제거
0      → 최댓값 1 출력 및 제거
3      → 힙에 3 추가
2      → 힙에 2 추가
1      → 힙에 1 추가
0      → 최댓값 3 출력 및 제거
0      → 최댓값 2 출력 및 제거
0      → 최댓값 1 출력 및 제거
0      → 힙이 비어있으므로 0 출력
0      → 힙이 비어있으므로 0 출력
```

➡️ **최대 힙 자료구조**를 구현하거나 Java의 `PriorityQueue`를 활용하여 해결하는 문제이다.

---

## 🧠 해결 전략 개요

* Java의 `PriorityQueue`를 사용하되, 최대 힙을 만들기 위해 `Comparator.reverseOrder()`를 사용한다.
* 또는 배열 기반으로 최대 힙을 직접 구현할 수 있다.
* 각 연산:
  * 자연수 입력: `heap.offer(x)` (또는 직접 구현한 `push(x)`)
  * 0 입력: `heap.isEmpty() ? 0 : heap.poll()` (또는 직접 구현한 `pop()`)

---

## ✨ 예시

### 예시 1: 문제에서 제공한 예제

**입력**
```
13
0
1
2
0
0
3
2
1
0
0
0
0
0
```

**출력**
```
0
2
1
3
2
1
0
0
```

**설명**: 
* 첫 번째 0: 힙이 비어있으므로 0 출력
* 1, 2 추가: 힙에 [2, 1] 저장 (2가 루트)
* 두 번째 0: 최댓값 2 출력 및 제거, 힙에 [1] 남음
* 세 번째 0: 최댓값 1 출력 및 제거, 힙이 비어있음
* 3, 2, 1 추가: 힙에 [3, 2, 1] 저장 (3이 루트)
* 네 번째 0: 최댓값 3 출력 및 제거, 힙에 [2, 1] 남음
* 다섯 번째 0: 최댓값 2 출력 및 제거, 힙에 [1] 남음
* 여섯 번째 0: 최댓값 1 출력 및 제거, 힙이 비어있음
* 일곱 번째, 여덟 번째 0: 힙이 비어있으므로 0 출력

---

## 📝 정리

이 문제는 **최대 힙 자료구조**를 구현하거나 활용하는 문제로, 우선순위 큐를 사용하여 효율적으로 해결할 수 있다. 

Java의 `PriorityQueue`는 기본적으로 최소 힙이므로, 최대 힙을 만들기 위해 `Comparator.reverseOrder()`를 사용해야 한다.
