# 나눌 수 있는 부분 수열 (BOJ 3673) 문제 요약

## 📌 문제 요약

양의 정수로 이루어진 수열이 주어진다.
연속하는 부분 수열의 합 중에서 `d`로 나누어 떨어지는 경우의 수를 구하는 문제다.

즉, 모든 구간 `[l, r]`에 대해 부분합 `A[l] + ... + A[r]`가 `d`의 배수인 구간 개수를 출력하면 된다.

테스트 케이스는 여러 개 주어진다.

---

## 📥 입력

- 첫째 줄: 테스트 케이스 수 `c` (`1 <= c <= 200`)
- 각 테스트 케이스:
- 첫째 줄: `d`, `n` (`1 <= d <= 1,000,000`, `1 <= n <= 50,000`)
- 둘째 줄: 수열 원소 `n`개 (`1 <= Ai <= 1,000,000,000`)

---

## 📤 출력

- 각 테스트 케이스마다,
- 연속 부분 수열의 합이 `d`로 나누어 떨어지는 구간의 개수를 한 줄에 출력

---

## ✨ 예시

- `d = 7`, 수열 `[1, 2, 3]` -> 조건을 만족하는 구간 수: `0`
- `d = 4`, 수열 `[2, 1, 2, 1, 1, 2, 1, 2]` -> 조건을 만족하는 구간 수: `6`
