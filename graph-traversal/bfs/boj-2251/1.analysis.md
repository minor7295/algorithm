# 물통

## 1. 문제 요약

용량이 각각 `A`, `B`, `C`인 세 물통이 있습니다. 처음에는 A와 B는 비어 있고, C만 가득 차 있습니다. 한 번의 동작에서 한 물통의 물을 다른 물통으로 옮길 수 있으며, 출발 물통이 비거나 도착 물통이 가득 찰 때까지 연속해서 붓습니다. 물의 손실은 없습니다.

문제의 목표는 가능한 모든 동작을 반복했을 때, **A 물통이 비어 있는 상태에서 C 물통이 가질 수 있는 물의 양**을 모두 구하는 것입니다.

---

## 2. 입력 조건 정리

- 입력: `A B C`
- 범위: `1 <= A, B, C <= 200`

초기 상태는 항상 `(a, b, c) = (0, 0, C)`입니다.

---

## 3. 출력 조건 정리

- A 물통이 0인 모든 상태에서 C 물통의 물의 양을 수집합니다.
- 수집한 값을 오름차순으로 공백 구분해 출력합니다.

---

## 4. 상태 관점에서의 문제 해석

각 시점은 `(a, b, c)`라는 하나의 상태로 표현할 수 있습니다.

- `a`: A 물통의 현재 물의 양
- `b`: B 물통의 현재 물의 양
- `c`: C 물통의 현재 물의 양

문제는 시작 상태 `(0, 0, C)`에서 출발해 도달 가능한 상태들 중 `a == 0`인 상태의 `c` 값들을 모으는 작업으로 해석할 수 있습니다.

---

## 5. 확인해야 할 핵심 포인트

- 물의 총량은 항상 일정합니다. (`a + b + c = C`)
- 한 번의 붓기에서 가능한 결과는 "출발이 빈다" 또는 "도착이 찬다" 두 경우뿐입니다.
- 정답은 단일 값이 아니라 가능한 값들의 집합입니다.
