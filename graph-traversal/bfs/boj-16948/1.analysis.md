# 데스 나이트

## 1. 문제 요약

`N x N` 체스판에서 데스 나이트가 시작점 `(r1, c1)`에서 목표점 `(r2, c2)`로 이동할 때, **최소 이동 횟수**를 구하는 문제입니다.

데스 나이트는 한 번에 아래 6가지 방향으로만 이동할 수 있습니다.
- `(-2, -1)`
- `(-2, +1)`
- `(0, -2)`
- `(0, +2)`
- `(+2, -1)`
- `(+2, +1)`

체스판 밖으로는 이동할 수 없습니다.

---

## 2. 입력 조건 정리

- 체스판 크기 `N` (`5 <= N <= 200`)
- 시작 좌표 `r1, c1`
- 도착 좌표 `r2, c2`
- 좌표는 모두 `0`부터 시작하는 인덱스입니다.

---

## 3. 출력 조건 정리

- 시작점에서 도착점까지의 최소 이동 횟수를 출력합니다.
- 도달할 수 없으면 `-1`을 출력합니다.
