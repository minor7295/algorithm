# 트리의 부모 찾기 (BOJ 11725) 문제 요약

## 📌 문제 요약

루트가 정해지지 않은 트리가 주어진다.
트리의 루트를 `1`번 노드로 정했을 때, 각 노드의 부모 노드를 구하는 문제다.

즉, `2`번 노드부터 `N`번 노드까지 각 노드의 직속 부모 번호를 순서대로 출력하면 된다.

---

## 📥 입력

- 첫째 줄: 노드의 개수 `N` (`2 <= N <= 100,000`)
- 둘째 줄부터 `N-1`개 줄: 트리에서 연결된 두 정점 `u`, `v`

---

## 📤 출력

- 첫째 줄부터 `N-1`개의 줄에,
- `2`번 노드부터 `N`번 노드까지 각 노드의 부모 노드 번호를 한 줄에 하나씩 출력

---

## ✨ 예시

- 예제 1 기준 부모 관계: `2->4`, `3->6`, `4->1`, `5->3`, `6->1`, `7->4`
- 따라서 출력은 `2`번부터 `7`번 노드의 부모를 순서대로 나열한 값이 된다.
