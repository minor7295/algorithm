# BOJ 2667 - 단지번호붙이기

## 문제 분석

### 문제 요약
- N×N 크기의 정사각형 지도가 주어짐
- 1은 집이 있는 곳, 0은 집이 없는 곳
- 상하좌우로 인접한 집들은 하나의 단지로 묶임 (대각선은 연결되지 않음)
- 총 단지 수와 각 단지에 속하는 집의 수를 오름차순으로 출력

### 입력 조건
- N: 5 ≤ N ≤ 25
- 각 줄에 N개의 0 또는 1이 입력됨

### 출력 조건
- 첫 번째 줄: 총 단지 수
- 이후 각 줄: 각 단지의 집 수 (오름차순)

### 예제 분석
```
입력:
7
0110100
0110101
1110101
0000111
0100000
0111110
0111000

출력:
3
7
8
9
```
