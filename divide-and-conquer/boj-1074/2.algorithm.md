# Z (BOJ 1074) 알고리즘

## 1) 풀이 아이디어
재귀 기반 분할 정복으로 풀이합니다.

크기 `2^N x 2^N` 정사각형을 매 단계에서 4등분하고, `(r, c)`가 속한 사분면을 찾습니다.
해당 사분면 이전에 방문되는 칸 수를 누적한 뒤, 좌표를 그 사분면의 로컬 좌표로 변환하여 재귀 호출합니다.

사분면 오프셋은 아래와 같습니다.
- UL: `0 * quarter`
- UR: `1 * quarter`
- LL: `2 * quarter`
- LR: `3 * quarter`

여기서
- `half = 2^(N-1)`
- `quarter = half * half`
입니다.

## 2) 재귀 절차
1. 베이스 케이스: `N == 1`이면 `2 x 2` 블록이므로 `r*2 + c`를 반환합니다.
2. `half`, `quarter`를 계산합니다.
3. `(r, c)`가 속한 사분면을 판별합니다.
4. 사분면 오프셋을 더한 뒤, 좌표를 로컬 좌표로 줄여 `N-1`로 재귀 호출합니다.

## 3) 복잡도
- 시간 복잡도: `O(N)`입니다.
- 공간 복잡도: `O(N)`입니다. (재귀 호출 스택)

`N <= 15`이므로 재귀 깊이가 최대 15에 불과하여 충분히 안전합니다.
