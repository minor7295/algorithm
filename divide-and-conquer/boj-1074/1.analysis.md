# Z (BOJ 1074) 문제 분석

## 1) 문제 요약
크기가 `2^N x 2^N`인 2차원 배열을 Z 순서로 방문할 때, `(r, c)` 위치가 몇 번째(0-based)로 방문되는지 출력하는 문제입니다.

## 2) 입력 조건
- 첫째 줄에 정수 `N`, `r`, `c`가 주어집니다.
- `1 <= N <= 15`
- `0 <= r, c < 2^N`

## 3) 출력 조건
- `(r, c)`를 방문하는 순서를 출력합니다.

## 4) 핵심 포인트
- 정답은 방문 순서의 0-based 인덱스입니다.
- 배열 전체를 출력하거나 실제 방문 과정을 모두 시뮬레이션할 필요는 없습니다.
