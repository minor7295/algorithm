# 오큰수 (BOJ 17298) 문제 요약

## 📌 문제 요약

크기가 `N`인 수열 `A1..AN`이 주어진다.
각 원소 `Ai`마다, 오른쪽에 있으면서 `Ai`보다 큰 수 중 가장 왼쪽에 있는 값을 찾는다.
이 값을 오큰수(`NGE(i)`)라고 한다.

해당 값을 찾을 수 없으면 `-1`을 출력한다.
최종적으로 `NGE(1)`부터 `NGE(N)`까지를 공백으로 구분해 출력한다.

---

## 📥 입력

- 첫째 줄: 수열의 크기 `N` (`1 <= N <= 1,000,000`)
- 둘째 줄: 수열 원소 `A1 A2 ... AN` (`1 <= Ai <= 1,000,000`)

---

## 📤 출력

- `N`개의 오큰수 값을 공백으로 구분해 한 줄에 출력
- 존재하지 않는 경우 해당 위치의 값은 `-1`

---

## ✨ 예시

- `A = [3, 5, 2, 7]` -> `5 7 7 -1`
- `A = [9, 5, 4, 8]` -> `-1 8 8 -1`
