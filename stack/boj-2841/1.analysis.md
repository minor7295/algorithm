# 외계인의 기타 연주 (BOJ 2841) 문제 분석

## 1) 문제 요약
기타 6개 줄 각각에서 누르고 있는 프렛 상태가 주어질 때,
멜로디 순서대로 음을 연주하기 위해 손가락을 누르거나 떼는 최소 횟수를 구하는 문제입니다.

각 음은 `(줄 번호, 프렛 번호)`로 주어지고,
해당 줄에서 가장 높은 프렛이 실제 소리가 됩니다.

## 2) 입력 조건
- 첫째 줄: `N`, `P`
- `1 <= N <= 500,000`
- `2 <= P <= 300,000`
- 다음 `N`줄: `line`, `fret`

실제 기타 줄은 1번부터 6번까지이므로 줄별 상태를 독립적으로 관리할 수 있습니다.

## 3) 출력 조건
- 멜로디를 모두 연주할 때 필요한 최소 손가락 움직임 횟수를 출력합니다.
