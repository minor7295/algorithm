# 가스관 (BOJ 2931) 문제 분석

## 1) 문제 상황 요약
러시아 가스를 크로아티아로 보내기 위해 `M`(모스크바)에서 `Z`(자그레브)까지 이어지는 가스관 지도가 주어집니다.
지도는 `R x C` 격자이며, 각 칸에는 가스관 블록이 있거나 비어 있습니다.

가스는 가스관 블록을 따라 흐르며, 블록 연결이 맞아야 다음 칸으로 전달됩니다.
두 도시가 설계를 마친 뒤 해커가 블록 하나를 지워서 해당 칸이 빈칸(`.`)으로 바뀌었습니다.

문제의 목표는 다음 두 가지를 찾는 것입니다.

1. 해커가 지운 블록의 정확한 위치(행, 열)
2. 그 위치에 원래 있던 블록의 종류

---

## 2) 지도에 등장하는 문자
입력 지도에는 아래 문자들이 사용됩니다.

- `.` : 빈칸
- `|` : 상하로 연결되는 블록
- `-` : 좌우로 연결되는 블록
- `+` : 상하좌우 모두 연결 가능한 블록
- `1`, `2`, `3`, `4` : 방향이 다른 꺾임 블록
- `M` : 모스크바 위치
- `Z` : 자그레브 위치

문제에서 `+` 블록은 특히 중요하며, 상황에 따라 수직/수평 양쪽 흐름을 모두 처리해야 합니다.

---

## 3) 입력 형식 요약
- 첫 줄에 격자 크기 `R`, `C`가 주어집니다. (`1 <= R, C <= 25`)
- 다음 `R`개의 줄에 길이 `C`의 문자열이 주어집니다.
- 각 문자는 위에서 정의한 문자 집합 중 하나입니다.
- `M`, `Z`는 각각 정확히 한 번씩 등장합니다.

즉, 한 장의 완성된 가스관 지도에서 블록 하나가 사라진 상태가 입력으로 제공됩니다.

---

## 4) 출력 형식 요약
출력은 한 줄이며, 다음 3개 값을 공백으로 구분해 출력합니다.

1. 지워진 블록의 행 번호
2. 지워진 블록의 열 번호
3. 지워진 블록 문자

행과 열 번호는 문제의 일반 격자 출력 규칙대로 **1부터 시작**합니다.

---

## 5) 문제에서 요구하는 정답의 의미
이 문제에서 정답은 단순히 빈칸 좌표를 찾는 것이 아니라,
"그 빈칸에 어떤 블록이 있어야 원래 가스관 시스템이 성립하는지"까지 함께 복원하는 결과입니다.

따라서 최종 답은 위치 정보와 블록 종류를 동시에 포함해야 하며,
입력 예시의 출력 형태도 항상 `행 열 블록`의 3요소로 구성됩니다.
