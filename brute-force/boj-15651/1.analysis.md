# N과 M (3)

## 📌 문제 요약

자연수 N과 M이 주어졌을 때, 1부터 N까지의 자연수 중에서 **M개를 중복을 허용하여** 선택하여 만들 수 있는 모든 수열을 사전 순으로 출력하는 프로그램을 작성한다.

---

## 🔍 문제 설명

* 1부터 N까지의 자연수 중에서 **M개를 선택**하여 수열을 만든다
* **같은 수를 여러 번 골라도 된다** (중복 허용)
* 만들어진 수열은 **사전 순으로 증가하는 순서**로 출력해야 한다
* 중복되는 수열을 여러 번 출력하면 안 된다

---

## 📥 입력 조건

* 첫째 줄에 자연수 `N`, `M`이 주어진다
  * `1 ≤ M ≤ N ≤ 7`

---

## 📤 출력 조건

* 한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다
* 각 수열은 공백으로 구분해서 출력해야 한다
* 수열은 **사전 순으로 증가하는 순서**로 출력해야 한다
* 중복되는 수열을 여러 번 출력하면 안 된다

---

## 💡 문제 핵심 해석

이 문제는 **중복 순열(permutation with repetition)** 문제이다.

* 각 자리마다 1부터 N까지의 모든 수를 선택할 수 있다
* 같은 수를 여러 번 선택해도 된다
* 총 가능한 수열의 개수는 **N^M** 개이다

예를 들어, N=3, M=2인 경우:
* 각 자리마다 3가지 선택지가 있으므로
* 총 3^2 = 9개의 수열이 생성된다

➡️ **백트래킹(Backtracking)** 또는 **재귀(Recursion)**를 사용하여 모든 경우를 탐색하는 문제이다.

---

## 🧠 해결 전략 개요

* 재귀 함수를 사용하여 각 자리에 1부터 N까지의 수를 하나씩 선택한다
* M개의 자리를 모두 채우면 결과를 출력한다
* 사전 순으로 출력하기 위해 각 자리에서 1부터 N까지 순서대로 선택한다
* 중복이 허용되므로 별도의 방문 체크가 필요 없다

---

## ✨ 예시

### 예시 1: N=3, M=1

**입력**
```
3 1
```

**출력**
```
1
2
3
```

**설명**: 각 자리에 1, 2, 3 중 하나를 선택. 총 3개의 수열.

---

### 예시 2: N=4, M=2

**입력**
```
4 2
```

**출력**
```
1 1
1 2
1 3
1 4
2 1
2 2
2 3
2 4
3 1
3 2
3 3
3 4
4 1
4 2
4 3
4 4
```

**설명**: 
* 첫 번째 자리에 1을 선택하면, 두 번째 자리에 1, 2, 3, 4를 각각 선택
* 첫 번째 자리에 2를 선택하면, 두 번째 자리에 1, 2, 3, 4를 각각 선택
* ... (반복)
* 총 4^2 = 16개의 수열

---

### 예시 3: N=3, M=3

**입력**
```
3 3
```

**출력**
```
1 1 1
1 1 2
1 1 3
1 2 1
1 2 2
1 2 3
1 3 1
1 3 2
1 3 3
2 1 1
2 1 2
2 1 3
2 2 1
2 2 2
2 2 3
2 3 1
2 3 2
2 3 3
3 1 1
3 1 2
3 1 3
3 2 1
3 2 2
3 2 3
3 3 1
3 3 2
3 3 3
```

**설명**: 
* 각 자리마다 3가지 선택지가 있으므로
* 총 3^3 = 27개의 수열이 생성된다
* 사전 순으로 정렬되어 출력된다

---

## 📝 정리

이 문제는 **중복을 허용하는 순열 생성** 문제로, 백트래킹을 사용하여 모든 경우를 탐색하는 전형적인 문제이다. 

각 자리에서 1부터 N까지 순서대로 선택함으로써 자연스럽게 사전 순으로 출력되며, 중복이 허용되므로 별도의 방문 체크 없이 모든 경우를 탐색할 수 있다.
